# NovaCart - Responsive E-Commerce Portfolio Website
NovaCart is a modern fully responsive e-commerce portfolio website built using HTML, CSS, and JavaScript. This project is designed for frontend portfolio showcase purposes and demonstrates responsive layouts, product filtering, modern UI/UX design, and interactive web components.
---
## Features
- Fully Responsive Design
- Modern Ecommerce UI
- Hero Section with CTA Buttons
- Sticky Navigation Bar
- Product Categories Section
- Dynamic Product Filtering
- Featured Products Grid
- Testimonials Section
- Newsletter Subscription Section
- Smooth Scroll Animations
- Mobile Friendly Navigation Menu
- Interactive Buttons and Hover Effects
- Portfolio Ready Design
---
## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts
- Unsplash Images
